PolarsysMaturity

This repository is for development of tools and data used in the Maturity Assessment task force of the PolarSys working group. The full quality model, along with all files needed to understand and execute it, are described on the [PolarSys wiki](https://polarsys.org/wiki/EclipseQualityModel).

Quality model visualisation

This javascript code allows to visualise the quality model being defined for the maturity assessment process. Please note that any remark on the quality model itself should be sent to PolarSys mailing lists, not GitHub pages.
