PolarsysMaturity

This repository is for development of tools used in the Maturity Assessment task force of the PolarSys working group.

Demonstration

Check the following samples:

  • Main quality attributes only: polarsys_qm_simple.html
  • All quality attributes: polarsys_qm_medium.html
  • All quality attributes and concepts description: polarsys_qm_full.html

Quality model visualisation

This javascript code allows to visualise the quality model being defined for the maturity assessment process. Please note that any remark on the quality model itself should be sent to PolarSys mailing lists, not GitHub pages.
